# tree
打印文件夹结构

## Usage

```
Usage: Tree [options]

    -p, --path:
        Path to the project that you want tree

    -d, --depth:
        Depth to the tree

    --ignore:
        Ignore to the dir in tree

    -h, --help:
        Prints a help message.

```

## Example

```shell
python tree.py
```

```
log >>
└── tree
    ├── .gitignore
    ├── LICENSE
    ├── README.md
    └── tree.py
```
